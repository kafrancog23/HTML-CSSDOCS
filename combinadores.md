# ☮ Combinadores



Nos permiten combinar múltiples selectores y crear una mayor especificidad.

#### Hermano adyacente o cercano (Adjacent sibling)

```css
div + p {
	...
}
```

#### Hermano general (General sibling)

```css
div ~ p {
	...
}
```

#### Hijo (Child)

```css
div > p {
	...
}
```

#### Descendiente (Descendant)

```css
div p {
	...
}
```
